# Ness Plumbing & Heating Site

Static marketing website for Ness Plumbing & Heating, focused on:

- Air source heat pump servicing
- Emergency plumbing
- General plumbing
- Hull and East Yorkshire service coverage

## Project Structure

- `index.html` - main site file
- `assets/` - logos and image assets
- `.gitignore` - ignored local/system files
- `LICENSE` - usage rights for this project

## Local Editing

This site is a simple static build with no framework or package install required.

To work on it locally:

1. Open the project folder.
2. Edit `index.html`.
3. Open the file in a browser to preview changes.

## Git Workflow

Typical update flow:

```powershell
git add .
git commit -m "Describe your change"
git push
```

## Deployment

This site is designed to be deployed as a static site on Render.

Recommended Render settings:

- Service type: `Static Site`
- Branch: `main`
- Build command: leave blank
- Publish directory: `.`

## Live Business Focus

The current site messaging is weighted toward seasonal priorities:

- Air source heat pump servicing
- Emergency plumbing
- General plumbing

Gas-related work is still referenced where relevant, but is not the main emphasis in the current version.

## Notes
